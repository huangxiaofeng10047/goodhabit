# goodhabit
一个问答小测试。nodejs + mongodb + H5动画

###开启mongodb

安装mongodb后，进入mongodb安装目录下的bin，终端输入
```
./mongod
```

另一个终端输入
```
./mongo
```
mongodb几个常用命令
```
mongo
show dbs
use habit
show collections
db.habits.find({})
```

###启动
```
npm start
前台：http://localhost:8090/goodhabit/
简单后台：http://localhost:8090/goodhabit/admin
```
